# Laboratorios del Curso 4: Ingeniería de Software con IA

Este directorio contiene los laboratorios prácticos del Curso 4 del programa AAIMC.

## Estructura

| Carpeta | Módulo | Descripción |
|---------|--------|-------------|
| `M3_Git` | Módulo 3: Git y Control de Versiones | Laboratorio de flujo de trabajo Git profesional |
| `M4_APIs` | Módulo 4: APIs y Backend | Laboratorio de consumo de APIs con Python |
| `M5_Databases` | Módulo 5: Bases de Datos | Laboratorio de SQL y ORM con SQLAlchemy |
| `M8_AI_Integration` | Módulo 8: Integración de IA | Laboratorio de SDKs de OpenAI y Anthropic |

## Instrucciones Generales

1. Clona este repositorio: `git clone https://github.com/LaloLalo1999/AAIMC_04_intro_to_github.git`
2. Navega a la carpeta del laboratorio correspondiente.
3. Sigue las instrucciones en el archivo `README.md` de cada laboratorio.
4. Crea una rama para tu trabajo: `git checkout -b lab-m[X]-[tu-nombre]`
5. Realiza commits frecuentes y descriptivos.
6. Sube tu rama al repositorio: `git push origin lab-m[X]-[tu-nombre]`
